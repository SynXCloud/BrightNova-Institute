# APEX Academy - Gamified EdTech Platform

![APEX Academy Logo](https://img.shields.io/badge/APEX-Academy-F5A623?style=for-the-badge&logo=codeforces&logoColor=black)
![Version](https://img.shields.io/badge/version-1.0.0-blue?style=for-the-badge)

A premium, high-performance coaching institute landing page built with a **"Dark Academic Gamified"** aesthetic. The platform is designed to convert visitors into enrolled students using interactive elements, dynamic tracking, and instant feedback loops.

## ✨ Features

- **Gamified User Experience**: Earn XP (Experience Points) and unlock achievements by interacting with the site (scrolling, clicking tabs, submitting forms).
- **Interactive Mock Test Platform**: Fully functional built-in mock test with timer, question palette, and instant result visualization.
- **Dynamic Performance Dashboard**: A real-time student tracking visualization showcasing the institute's tech-forward approach with circular progress rings and bar charts.
- **Scholarship Eligibility Calculator**: A 4-step interactive flow to calculate potential scholarship discounts based on user inputs.
- **3D Faculty Grid**: Hover-flip cards revealing the immense experience and success rates of the faculty.
- **Fully Responsive & Performant**: Built entirely without heavy frontend frameworks or external JavaScript libraries.

## 🛠️ Tech Stack

- **HTML5**: Semantic and highly structured.
- **Vanilla CSS3**: Advanced grid/flexbox layouts, custom variables (tokens), animations, and a responsive mobile-first approach.
- **Vanilla JavaScript**: DOM manipulation, Intersection Observers for scroll animations, state management for mock tests and forms, and local session gamification.
- **External Assets**: 
  - [Google Fonts](https://fonts.google.com/) (Syne, Nunito, JetBrains Mono)
  - [Font Awesome](https://fontawesome.com/) (Icons)

## 🚀 Getting Started

Simply open `index.html` in any modern web browser to view the application locally. No build tools or package managers are required!

```bash
# Clone the repository
git clone https://github.com/SynXCloud/BrightNova-Institute.git

# Navigate to the directory
cd BrightNova-Institute

# Open in your default browser (Mac)
open index.html

# Open in your default browser (Linux)
xdg-open index.html
```
